关于旧版本

Vue CLI 的包名称由 `vue-cli` 改成了 `@vue/cli`。
如果你已经全局安装了旧版本的 `vue-cli` (1.x 或 2.x)，你需要先通过 `npm uninstall vue-cli -g` 或 `yarn global remove vue-cli` 卸载它。
